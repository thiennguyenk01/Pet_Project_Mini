▶ 3.1 If

if được sử dụng khi bạn cần kiểm tra điều kiện trước khi thực hiện một hoặc nhiều câu lệnh. Các câu lệnh bên trong if phải thụt lề.

Cú pháp

Cú pháp
if condition:
    # code

Nếu condition có giá trị True thì code bên trong if được thực hiện, ngược lại nếu False thì bỏ qua.

Python
if 100 > 50:
    print('hello hawkdev !')
Kết quả
hello hawkdev !

Kiểm tra số chẵn, lẻ

Python
n = 200
if n % 2 == 0:
    print(n, 'is even')

m = 201
if m % 2 == 1:
    print(m, 'is odd')
Kết quả
200 is even
201 is odd

Kiểm tra số nằm trong đoạn [a, b]

Python
a, b = 50, 100
m = 70
if (m >= a) and (m <= b):
    print('YES')
Kết quả
YES

▶ 3.2 Else

else được sử dụng trong trường hợp condition bên trong if là sai.

Cú pháp

Cú pháp
if condition:
    # code if condition is True
else:
    # code if condition is False
Python
n = 100
if n % 2 == 0:
    print('Even')
else:
    print('Odd')
Kết quả
Even

▶ 3.3 Elif

elif (else if) được sử dụng bên dưới if để kiểm tra thêm điều kiện bổ sung. Nếu không có điều kiện nào đúng thì khối lệnh trong else được thực thi.

Cú pháp

Cú pháp
if condition1:
    # code1
elif condition2:
    # code2
...
elif conditionN:
    # codeN
else:
    # codeElse
Python
a, b = 100, 200
if a < b:
    print(a, 'less than', b)
elif a == b:
    print(a, 'equal to', b)
else:
    print(a, 'greater than', b)
Kết quả
100 less than 200

▶ 3.4 Shorthand if và toán tử ba ngôi

Shorthand if (if trên 1 dòng)

Bạn có thể sử dụng câu lệnh if trên 1 dòng.

Python
a, b = 100, 200
if a < b: print(a, 'less than', b)
Kết quả
100 less than 200

Nếu trong if có nhiều câu lệnh, có thể đặt dấu chấm phẩy giữa các câu lệnh.

Python
a, b = 100, 200
if a < b: print(a, 'less than', b); print('hawkdev'); print('abcd')
Kết quả
100 less than 200
hawkdev
abcd

Toán tử ba ngôi (Ternary Operator)

Cú pháp: variable = statement_if_True if condition else statement_if_False

Python
a, b = 100, 200
res = 'hawkdev' if a < b else 'python'
print(res)
Kết quả
hawkdev

▶ 3.5 If lồng nhau (Nested if)

Khi điều kiện quá phức tạp, bạn có thể sử dụng if lồng nhau để kiểm tra từng điều kiện một.

Bài toán

Kiểm tra N xem có phải là số nằm trong đoạn [20, 30] và chia hết cho ít nhất một trong ba số 2, 3, 5.

Cách 1: Điều kiện kết hợp

Python
n = 27
if n >= 20 and n <= 30 and (n % 2 == 0 or n % 3 == 0 or n % 5 == 0):
    print('YES')
else:
    print('NO')
Kết quả
YES

Cách 2: Nested if

Python
n = 27
if n >= 20 and n <= 30:
    if n % 2 == 0 or n % 3 == 0 or n % 5 == 0:
        print('YES')
    else:
        print('NO')
else:
    print('NO')
Kết quả
YES

4.1 Lambda là gì?

Lambda là một cách đơn giản để định nghĩa một hàm trong Python. Lambda thường được gọi dưới cái tên toán tử lambda hoặc hàm lambda.

🎨 1.5 Sắp xếp nested list theo nhiều tiêu chí bằng hàm tự định nghĩa

✖ 4.3 Nhân hai ma trận

Giả sử có 2 ma trận a cỡ nxm, ma trận b cỡ pxq, để ma trận a có thể nhân với ma trận b thì số cột của ma trận a, tức là m phải bằng số hàng của ma trận b, tức là p.

📎
a[n][m] x b[p][q] = c[n][q] — khi m = p thì ma trận tích của a với b sẽ là ma trận c có cỡ nxq.

Phần tử ở chỉ số (i, j) của ma trận tích c được tính bằng cách nhân từng cặp phần tử ở hàng i của ma trận a với các phần tử ở cột j của ma trận b.

Python
n, m, p = map(int, input().split())
a = [0] * n
b = [0] * m
for i in range(n):
    a[i] = list(map(int, input().split()))
for i in range(m):
    b[i] = list(map(int, input().split()))
c = [[0 for _ in range(p)] for _ in range(n)]
for i in range(n):
    for j in range(p):
        for k in range(m):
            c[i][j] += a[i][k] * b[k][j]
print(c)
⚠️
Độ phức tạp của nhân 2 ma trận bằng vòng lặp 3 tầng là O(n x m x p). Với n, m, p lớn, cần lưu ý giới hạn thời gian.

    animal_sound(animal)

import math

class Fraction:
    def __init__(self, numerator, denominator):
        if denominator == 0:
            raise ValueError("Denominator cannot be zero")
        gcd = math.gcd(numerator, denominator)
        self.num = numerator // gcd
        self.den = denominator // gcd

    def __add__(self, other):
        return Fraction(self.num * other.den + other.num * self.den,
                        self.den * other.den)

    def __sub__(self, other):
        return Fraction(self.num * other.den - other.num * self.den,
                        self.den * other.den)

    def __mul__(self, other):
        return Fraction(self.num * other.num, self.den * other.den)

    def __eq__(self, other):
        return self.num == other.num and self.den == other.den

    def __lt__(self, other):
        return self.num * other.den < other.num * self.den

    def __str__(self):
        return f"{self.num}/{self.den}"

    def __repr__(self):
        return f"Fraction({self.num}, {self.den})"

    def to_float(self):
        return self.num / self.den

f1 = Fraction(1, 2)
f2 = Fraction(1, 3)
f3 = Fraction(2, 4)

print(f"{f1} + {f2} = {f1 + f2}")
print(f"{f1} * {f2} = {f1 * f2}")
print(f"{f1} == {f3}: {f1 == f3}")
print(f"{f2} < {f1}: {f2 < f1}")
print(f"{f1} as float: {f1.to_float()}")

class Shape(ABC):
    @abstractmethod
    def area(self):
        pass

    @abstractmethod
    def perimeter(self):
        pass

    def perimeter(self):
        return 2 * 3.14159 * self.radius

    def __init__(self, width, height):
        self.width = width
        self.height = height

    def area(self):
        return self.width * self.height

    def perimeter(self):
        return 2 * (self.width + self.height)

🔍 4.10 isinstance() và issubclass() trong kế thừa nhiều mức

Python

    pass

class Dog(Animal):
    pass

class Puppy(Dog):
    pass

p = Puppy()

print(isinstance(p, Puppy))     # True
print(isinstance(p, Dog))       # True (Puppy kế thừa Dog)
print(isinstance(p, Animal))    # True (Puppy → Dog → Animal)

print(issubclass(Puppy, Dog))      # True
print(issubclass(Puppy, Animal))   # True
print(issubclass(Dog, Puppy))      # False
